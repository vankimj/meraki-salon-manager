# Tap to Pay marketing assets

Apple requires the awareness splash + any Tap-to-Pay marketing imagery to use
**Apple-approved assets from the Tap to Pay on iPhone Marketing Toolkit** — you
cannot design your own. Get them from the link in your developer-program grant
email, or https://developer.apple.com/tap-to-pay/marketing-guidelines/

To use one: **replace `splash.png`** in this folder with the approved image
(keep the same filename). No code change needed. The dev-only "replace this
asset" hint disappears automatically in production/Release builds.
